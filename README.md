# 🖱️ 键鼠模拟工具

一个基于 Python 的键盘鼠标自动化工具，提供**宏录制回放**、**自动点击器**、**手动键鼠操作**等功能，适用于日常办公自动化、软件测试、重复操作解放双手等场景。

## 功能一览

| 功能 | 说明 |
|------|------|
| 🔴 **宏录制** | 录制鼠标点击和键盘按键操作，支持实时事件预览 |
| ▶️ **宏回放** | 按录制顺序回放操作，支持 **0.1x ~ 5x 倍速** 调节 |
| 💾 **宏保存** | 保存录制内容到本地，支持命名管理、加载、删除 |
| 🖱 **自动点击器** | 指定间隔（0.01s~10s）的连续自动点击，可随时启停 |
| 🎯 **手动操作** | 单击 / 双击 / 右键 / 滚动 / 文本输入 / 按键模拟 / Ctrl+C / Ctrl+V |
| 📸 **截图** | 一键截取全屏，自动保存到本地 |
| 🖥 **鼠标位置** | 实时显示鼠标当前坐标 |

## 安装

### 环境要求

- Python 3.6+
- Windows / macOS / Linux

### 安装步骤

```bash
# 下载项目
git clone https://github.com/guohaowei131700/-keymouse-tool.git
cd keymouse-tool

# 安装依赖
pip install -r requirements.txt

# 启动
python main.py
```

## 使用指南

### 录制宏

1. 点击 **「🔴 开始录制」**
2. 执行你要录制的操作（鼠标点击 / 键盘按键）
3. 点击 **「⏹ 停止录制」**
4. 右侧事件列表会显示录制的所有事件

### 回放宏

1. 录制完成后（或从列表加载已保存的宏）
2. 拖动 **回放速度** 滑块调节速度（1.0x = 原速）
3. 点击 **「▶️ 回放录制」**
4. 如需中途停止，点击 **「⏹ 停止回放」**

### 保存与加载

1. 录制完成后，在 **宏名称** 输入框填写名称
2. 点击 **「💾 保存宏」**
3. 下次启动程序时，点击宏列表中的项目即可加载

### 自动点击器

1. 设置 **间隔时间**（秒）
2. 点击 **「🖱 开始自动点击」**
3. 鼠标会按设定间隔连续点击当前位置
4. 再次点击同一按钮停止

### 手动操作

右侧面板提供即时的键鼠操作按钮，无需录制即可执行：
- **鼠标**：单击、双击、右键、滚动上下
- **键盘**：输入文本、模拟按键、Ctrl+C / Ctrl+V
- **快捷**：全屏截图、全选删除

## 常见问题

**Q: 录制的内容保存在哪里？**
A: 保存在项目目录下的 `config/macros.json` 文件中。

**Q: 回放速度不可调？**
A: 调节回放速度滑块后，重新点击回放即可生效。

**Q: 自动点击会点错位置？**
A: 自动点击会在**当前鼠标位置**执行。如需在固定位置点击，可先录制一次点击操作后回放。
