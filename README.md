# 🖱️ 键鼠模拟工具

一个基于 Python 的键盘鼠标模拟工具，支持宏录制回放、自动点击、快捷操作。

## 功能

- **宏录制 / 回放** — 录制鼠标点击和键盘操作，可调速回放
- **宏保存 / 加载** — 保存录制内容，随时调用
- **自动点击器** — 可配置间隔的自动鼠标点击
- **鼠标操作** — 单击、双击、右键、滚动
- **键盘操作** — 文本输入、按键模拟、快捷键
- **截图** — 一键截屏并复制路径

## 使用

```bash
pip install -r requirements.txt
python main.py
```

## 依赖

- pyautogui
- keyboard
- mouse
- pyperclip
